### Процесс обновления

| Действие | Частота | Кто |
|----------|---------|-----|
| Добавить данные спринта | После завершения спринта | Автоматически (MCP) |
| Пересчитать rolling average | После каждого спринта | Автоматически (MCP) |
| Проверить задачи без компонента | После каждого спринта | Аналитик + MCP |
| Обновить рекомендации | При изменении коэффициента > 20% | Автоматически (MCP) |

---

### Методология расчёта

```
Точность (%) = (Факт / Оценка) × 100%
Коэффициент = Факт / Оценка
Рекомендация (%) = (Коэффициент - 1) × 100%
Rolling Average = (Спринт 1 + Спринт 2 + ... + Спринт N) / N
```

**Статусы надёжности:**
| Статус | Спринтов | Надёжность | Использование |
|--------|----------|------------|---------------|
| 🔴 Критично | 1 | Низкая (только ориентир) | С осторожностью |
| 🟡 2 спринта | 2 | Средняя | Для планирования с буфером |
| 🟢 Надёжно | 3-5 | Высокая | Основной источник |
| ✅ Статистически значимый | 6+ | Очень высокая | Без дополнительных буферов |

### Интерпретация коэффициентов

| Коэффициент | Что значит | Как планировать |
|-------------|------------|-----------------|
| ×0.5-0.9 | Задачи выполняются быстрее оценки | Закладывать -10% к оценке |
| ×1.0-1.5 | Точная оценка | Без изменений |
| ×1.6-2.5 | Умеренный перерасход | Закладывать +50-150% к оценке |
| ×2.6-4.0 | Высокий перерасход | Закладывать +160-300% к оценке |
| ×4.1+ | Критический перерасход | Требуется декомпозиция + исследование |
